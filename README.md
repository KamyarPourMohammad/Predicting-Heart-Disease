# Predicting-Heart-Disease

⚠️ Work in Progress
Note: This project is currently under active development. The data preprocessing and exploratory analysis are underway, but the final model training and evaluation phases are not yet complete.
### Intro
Heart disease remains one of the leading causes of health complications globally. This project leverages modern data science to assist in the early detection and prevention of these illnesses, providing a digital tool to support healthcare efforts.
### Overview
The goal of this project is to implement a machine learning model capable of predicting the presence of heart disease with high accuracy.

The workflow involves:
1- Data Cleaning: Handling missing values and outliers.
2- Exploratory Data Analysis (EDA): Visualizing correlations and feature distributions.
3- Machine Learning: Training predictive models using scikit-learn.
